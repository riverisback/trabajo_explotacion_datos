# TRABAJO_EXPLOTACION_DATOS

Este proyecto forma parte del trabajo final del módulo de Explotación de Datos del programa MADE MBA. Su objetivo es aplicar técnicas de análisis exploratorio, manipulación de datos y visualización utilizando herramientas como Python, R y Excel, sobre un conjunto de datos sintético de clientes.

## Objetivos

- Realizar un análisis exploratorio de datos (EDA)
- Aplicar técnicas de limpieza y transformación
- Visualizar patrones relevantes para la toma de decisiones
- Documentar el proceso técnico y analítico

## Tecnologías utilizadas

- **Python**: pandas, matplotlib, seaborn, numpy  
- **Jupyter Notebook**: entorno interactivo para análisis en Python  

## Dataset

El archivo `dataset_sintetico_clientes.xlsx` contiene información ficticia de clientes, incluyendo variables demográficas, comportamiento de compra y segmentación. Se utiliza para ilustrar técnicas de explotación de datos en un contexto empresarial. No está disponible para descarga.

## Visualizaciones

Las imágenes en la carpeta `images/` representan gráficos generados durante el análisis.

## Requisitos

Para ejecutar el notebook proceder a instalar las dependencias del entorno Python descritas en el archivo `requirements.txt`.

Se recomienda usar un entorno virtual y tener instalado Jupyter Notebook.
